# The Base of the QAP Open Source Knowledgebase

<!-- ALL-CONTRIBUTORS-BADGE:START - Do not remove or modify this section -->
[![All Contributors](https://img.shields.io/badge/all_contributors-1-orange.svg?style=flat-square)](#contributors-)
<!-- ALL-CONTRIBUTORS-BADGE:END -->

The QAP Knowledgebase consists of these main pieces:
- [Website](https://base.qap.dev)
- [Wiki](https://github.com/qa-at-the-point/base/wiki) for links, docs, etc
- [Discussions](https://github.com/qa-at-the-point/base/discussions) for questions, comments, etc
- [Issues](https://github.com/qa-at-the-point/base/issues) for requests, bugs, etc
- [Contribute](https://github.com/qa-at-the-point/base/blob/main/CONTRIBUTING.md) to the Knowledgebase
- [Other repos](https://github.com/qa-at-the-point) in our GitHub organization

## Security

We use codeQL and dependabot alerts for vulnerabality analysis & fixes.

```sh
$ bundle audit
```
